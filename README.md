# clickhouse-replication-docker
Data Replication in ClickHouse (Docker Based Setup) , 3 Node Replication Cluster

## Requirement -- Download these files from Github repo

```
docker-compose.yaml
config.xml
clusters.xml
listen_host.xml
zookeeper.xml
macros_1.xml
macros_2.xml
macros_3.xml
```
